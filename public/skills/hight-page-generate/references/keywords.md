# 关键词速查清单

## CSS / 质感

| 关键词 | 作用 |
|--------|------|
| `backdrop-blur` | 玻璃/毛玻璃（Glassmorphism） |
| `soft-shadow` | 轻阴影，拉开层次 |
| `subtle-border` | 细边框，提升精致度 |
| `gradient-mesh` | 渐变网格，丰富背景层次 |
| `noise-texture` | 轻颗粒噪点，增加胶片感与质感 |
| `bento-grid` | 分块布局，信息分区清晰 |
| `responsive-grid` | 响应式网格，多端适配 |
| `sticky-section` | 粘性区块，滚动中保持关键信息 |
| `fluid-layout` | 流体布局，自适应更自然 |
| `scroll-snap` | 滚动吸附，分段切换更顺滑 |

## 动效

| 关键词 | 作用 |
|--------|------|
| `micro-interactions` | 微交互，小处见反馈 |
| `hover-feedback` | 悬停反馈，可点可交互更清晰 |
| `scroll-reveal` | 滚动显现，随滚动渐入 |
| `stagger-animation` | 错落/阶梯出现，制造层次与节奏 |
| `smooth-transition` | 平滑过渡，减少状态跳变 |
| `easing-curve` | 缓动曲线，运动更自然 |
| `parallax` | 视差，增强空间纵深 |
| `spring-animation` | 弹性动画，增加生命力（需克制） |

## Three.js / WebGL / anime.js

| 关键词 | 作用 |
|--------|------|
| `particle-field` | 粒子场，动态背景 |
| `3D-orb` | 3D 球体，科技视觉焦点 |
| `shader-gradient` | 着色器渐变，流动色彩 |
| `floating-geometry` | 漂浮几何体，增加层次 |
| `webgl-background` | WebGL 背景层 |
| `camera-parallax` | 相机/滚动视差 |
| `interactive-canvas` | 可交互画布（鼠标、滚动驱动） |

## 组件 / 系统

| 关键词 | 作用 |
|--------|------|
| `design-system` | 全局一致的设计规则 |
| `component-variants` | 变体（尺寸、语义色、主次按钮等） |
| `shadcn/ui` | 可拷贝、可定制的现代组件范式 |
| `radix-primitives` | 无头组件，行为与可访问性基础好 |
| `tailwind-css` | 原子化样式，快速一致落地 |
| `dashboard-layout` | 仪表盘式信息层级 |
| `command-palette` | 命令面板，效率型产品标配之一 |
| `modal` `toast` `empty-state` | 完整状态反馈 |

## 三档效果方向

| 方向 | 关键词组合建议 |
|------|---------------|
| **科技感** | 深色底 + gradient-mesh + soft-shadow + subtle-border + 可选 particle-field 背景 |
| **层次感** | noise-texture + backdrop-blur（玻璃卡）+ parallax + 可选 3D 点缀 |
| **产品感** | 浅色/中性色 + bento-grid + design-system + component-variants + 完整状态 |

> 使用方式：从每个分类中选 2-3 个关键词写入 Prompt，不要照单全收。
